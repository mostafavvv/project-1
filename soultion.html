Value Statement

DrySense's monitoring system removes the guessing of incontinence management. By notifying caregivers when a patient requires toileting services, the revolutionary technology dramatically reduces the burden of incontinence care while increasing patient wellbeing.

The DrySense device is embedded or attached to an individual's wheelchair cushion and monitors the local environment for temperature and humidity changes.

Device Description

Easy Installation

DrySense offers two intuitive installation methods. a. Install the device by fastening it to a wheelchair cushion under the seating cover. b. Install the device by embedding it into the wheelchair cushion.

Realtime monitoring

DrySense monitors the patient’s local environment for fluctuations in humidity and temperature every 30seconds. When a void event occurs, DrySense alerts caregivers simultaneously to ensure prompt care is received.

Hassle-Free

DrySense’s system is unintrusive and allows caregivers to move patients freely. The battery is long-lasting (3 days) and recharges within (x). Cleaning the device is only required if it’s in direct contact with void excrement (very rare given the device is under a seating cushion).
